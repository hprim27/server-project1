About This HTTP Server Project

Showcasing Backend Programming Skills

Project Goals

I wanted to create my own server for my portfolio to showcase backend programming skills. Instead of having the frontend make API calls directly to Wikipedia, I built a custom backend that handles the API integration.

  • Frontend-backend communication

API Endpoint Details

Endpoint: /api/artists

Method: GET

Purpose: Fetches musicians born on today's date

Process:

  1. Gets current date (month/day)
  2. Calls Wikipedia API: https://api.wikimedia.org/feed/v1/wikipedia/en/onthisday/births/{month}/{day}
  3. Filters results for music-related keywords
  4. Adds Wikipedia links for each artist
  5. Returns filtered JSON data

Music Artist Filtering

The server intelligently filters for musicians using keywords like:

  • musician, singer, rapper, band, composer
  • guitarist, drummer, bassist, pianist, violinist
  • producer, songwriter, vocalist, performer
